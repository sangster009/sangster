## How to use

- **Live presenting:** Open the site, go to `cover.html`, and use the matching script (e.g. `cover.md`) for that page. Move through the flow with the "Next" / "Start" links.
- **Rehearsal:** Read each script once, then run through the pages without reading to practice ad-libbing.
- **Customization:** Edit the markdown files to match your style, add timing notes, or drop sections you don’t need.

Each script includes:
- **Script:** Short paragraphs you can say verbatim or paraphrase.
- **Talking points:** Bullets for quick reference.
- **Transition:** What to say before moving to the next page.
